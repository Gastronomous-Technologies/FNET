FNET Overview

The FNET is Dual TCP/IPv4/IPv6 stack (under Apache v2 license) for building embedded communication software on 32bit MCUs.

The FNET stack features:
  • Bare-metal TCP/IP stack.
  • Non-blocking Socket layer API (STREAM, DGRAM, RAW).
  • Core Protocols:
    • TCP
    • UDP
    • IPv4, ICMPv4, IGMP, ARP
    • IPv6, ICMPv6, MLD, ND
  • Network Interfaces:
    • Ethernet
    • Loop-back
  • Services:
    • HTTP 1.0/0.9 Server:
      • GET, POST requests
      • CGI, SSI
      • Basic Access Authentication
      • HTTP over TLS (HTTPS)
    • Auto-IP service
    • TELNET server
    • DNS client
    • Link-Detection service
    • LLMNR server
    • MDNS server
    • DHCP client
    • TFTP server and client
    • TLS using mbedTLS library
    • PING service
    • ROM File System
    • Flash Memory Driver
    • Command Line Shell
    • Serial Driver
  • Tools:
    • GUI File System Generation Tool
    • GUI Throughput Benchmark Tool
  • Applications:
    • TFTP Boot-loader
    • Shell Application
    • Throughput Benchmark Application
